    NEngine.GLNSLCompiler. Scope

    new Scope()

    Represents a recursive Scope tree.

    The rootScope contains the cache of the temp variables used in intermediate operations (on translated code). This cache gets added to the begining during translation. It also holds functions and code automatically generated at the beginning (precode).

    ensureTypeCache(variable)

    Ensures that a given type has its cache variables instantiated for operations upon it, this is useful only during translation and final code writting, indexes them by codename.
    Adds them to variables array (avoid colisions) and cacheVariables (inform cache creation) sentence array. This affects only rootScope (only a single copy of each typecache is necessary)
