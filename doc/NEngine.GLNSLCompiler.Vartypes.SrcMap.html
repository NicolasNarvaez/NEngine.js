Class: SrcMap

NEngine.GLNSLCompiler.Vartypes. SrcMap

new SrcMap(src)

This was created in a moment of despair. Now i cant find a use to it...
Helps handling source mapping, stripping, scaping, etc. Provides functions to translate on its context (subsections of it, etc) like interpolate.

the level of source mapping it makes is low, doesnt create recursive SrcMap trees, but does create symbols trees
