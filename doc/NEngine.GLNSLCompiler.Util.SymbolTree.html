SymbolTree

NEngine.GLNSLCompiler.Util. SymbolTree

new SymbolTree(src, prefix)

This was created in a moment of despair. Now i cant find a use to it... mmmm, maybe just a middle-level tool
Helps handling source mapping, stripping, scaping, etc. Provides functions to translate on its context (subsections of it, etc) like interpolate.

the level of source mapping it makes is low, doesnt create recursive SymbolTree trees, but does create symbols trees (an internal shepherd would allow it, by indexing-trees, avoiding collisions on different symbol-tree tags)
