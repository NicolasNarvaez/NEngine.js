    NEngine.Media. AudioSourceSoundcloud

    new AudioSourceSoundcloud(opts)

    A Web Audio AudioNode wrapper for easy setting up a soundcloud connection inside an Audio element and loading a url. Requires the Soundcloud API to be loaded.
    Properties:
    Name Type Description
    cient_id String The Soundcloud app client_id, get one from the Soundcloud API docs
    ctx AudioContext Web Audio AudioContext
    tag Audio The Audio element used to generate the AudioNode using createMediaElementSource and handle the soundcloud connection/decoding.
    node AudioNode Use this in connect calls
    stream_url String Contains the strem url to use in the audio tag
    sound_obj Object The object Soundcloud returns when asked to resolve a url.
